# StopSigh

A time management application developed as part of CSE 300 
(Technical Communication) at Stony Brook University.
Team member: Anderson, Daniel C, Hailey, Ying Yi, Megan

## About
StopSigh is a student productivity app with a living mascot whose 
emotional state mirrors how well you're managing your time. It combines 
AI-powered task estimation, peer accountability, smart reminders, and 
optional focus monitoring into one emotionally expressive system.

## Features
- 📅 Brightspace & Calendar integration
- 👥 Accountability Contact
- ⚡ Energy system with emotional companion
- 🏆 Peer leaderboard
- 📷 Focus Mode with camera monitoring
- ⏰ Deadline tracking with live countdown

## Built With
- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started
Run `npm install` to install dependencies.  
Run `npm run dev` to start the development server.
